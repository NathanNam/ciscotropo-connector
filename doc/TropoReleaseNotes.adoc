The CiscoTropo connector facilitates connections between Mule integration and CiscoTropo applications. The CiscoTropo connector  provides to create new Tropo applications, delete applications, add/remove addresses (phone and SMS numbers, tokens), or view available exchanges (area codes and their associated regions).

=== Features

. Version 1.0 Supported Operations
The Cisco Tropo Connector 1.0 allows you to perform the following actions:

1.	Add Entity.
2.	Create Application.
3.	Create Session.
4.	Delete Application.
5.	Delete Address.
6.	Get Address.
7.	Get Address By Application
8.	Get Applications
9.	Get Available Prefixes
10.	Get Exchanges.
11.	Provisioning Application.
12.	Provisioning Addresses.
13.	Query Processor.
14.	Signal Operations.
